# Machine Learning Concepts From Scratch

Welcome to the **machine-learning-concepts** repository! This project is a hands-on, self-paced guide dedicated to implementing fundamental machine learning algorithms completely from scratch using Python and NumPy. 

Instead of relying on abstraction layers from high-level frameworks, these notebooks focus on building the direct mathematical, statistical, and optimization mechanics behind core ML models.

---

## 🚀 Repository Directory

The files in this repository are structured around core regression models, classification foundations, and gradient descent optimization strategies.

### 📈 Regression Models & Regularization
*   **`LR.ipynb`**: Baseline exploration and foundational concepts of Linear Regression.
*   **`MAKING MY OWN LR CLASS.ipynb`**: Step-by-step implementation of a custom object-oriented Linear Regression class from scratch.
*   **`MERA RIDGE REGRESSOR.ipynb`**: A complete walkthrough of Ridge Regression (L2 Regularization) solved using analytical/closed-form equations.
*   **`MeraRidgeGD.ipynb`**: Implementation of Ridge Regression optimized iteratively via Gradient Descent.

### 📉 Optimization Techniques (Gradient Descent Flavors)
*   **`GD LINEAR.ipynb`**: Applying classical Gradient Descent optimization directly to fit a linear regression framework.
*   **`MY BATCH GD REGRESSION.ipynb`**: Implementation of Batch Gradient Descent, performing parameter updates across the entire dataset concurrently.
*   **`mini-batch-gradient-descent-from-scratch.ipynb`**: Practical implementation of Mini-Batch Gradient Descent, striking a balance between calculation speed and convergence smoothness.
*   **`stochastic-gradient-descent-from-scratch.ipynb`**: Deep dive into Stochastic Gradient Descent (SGD) where weights are updated sample-by-sample for rapid learning.

### 🧪 Basic Classification
*   **`GD_LOGISTIC_R.ipynb`**: Binary classification from scratch using Logistic Regression optimized with Gradient Descent.
*   **`PRECEPTRON.ipynb`**: Implementation of the historical foundation of modern neural networks—the Perceptron learning algorithm.
